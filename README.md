# Meter Taxi

🚕 **The future of urban mobility** — MeterTaxi is a ride-hailing platform featuring multiple apps: Passenger, Driver, Partner, and Channel Owner.

The Passenger App launches **April 1, 2027**.

## 🚀 Project Structure

```text
/
├── public/                         # Static assets (favicon)
├── src/
│   ├── components/
│   │   ├── Header.astro            # Fixed navigation header
│   │   ├── CountdownSection.astro  # Hero with countdown to launch
│   │   ├── AppSection.astro        # App download cards
│   │   └── Footer.astro            # Footer
│   └── pages/
│       └── index.astro             # Main landing page
├── astro.config.mjs
├── package.json
└── tsconfig.json
```

## 🧞 Commands

| Command           | Action                                     |
| :---------------- | :----------------------------------------- |
| `npm install`     | Install dependencies                       |
| `npm run dev`     | Start local dev server at `localhost:4321` |
| `npm run build`   | Build production site to `./dist/`         |
| `npm run preview` | Preview build locally before deploying     |
| `npm run astro`   | Run Astro CLI commands                     |

## 📱 Apps

- **Passenger App** — Book rides instantly (launching April 1, 2027)
- **Driver App** — Earn on your schedule
- **Partner App** — Manage your fleet
- **Channel Owner App** — Own and operate your transport channel
